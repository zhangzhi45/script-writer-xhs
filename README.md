# script-writer-xhs

小红书文案撰写专家 Skill —— 支持选题→大纲→撰写→标题→评分全流程。

## 简介

这是一个为「麦克斯坦」账号定制的 AI Skill，专门用于撰写小红书口播/图文内容。它提供了一套完整的文案生产工作流，从选题策划到最终评分，确保内容质量稳定输出。

## 核心能力

| 功能 | 说明 |
|------|------|
| **选题驱动** | 根据主题搜索素材，提炼独特角度 |
| **大纲生成** | 基于 4A 结构模板生成内容框架 |
| **正文撰写** | 口语化风格，逻辑清晰、言之有物 |
| **润色优化** | 对初稿进行风格统一和表达优化 |
| **标题生成** | 基于 37 篇爆款数据分析的标题公式 |
| **量化评分** | 5 维度评分系统（满分 100） |

## 适用场景

- 开始写新内容（选题→大纲→撰写→标题→评分）
- 有素材/链接，需要提炼角度并撰写
- 有初稿需要润色优化
- 给现有文案打分评估
- 生成多个候选标题

## 使用方式

1. 将本仓库克隆到 WorkBuddy 的 skills 目录：
   ```bash
   cd ~/.workbuddy/skills/
   git clone https://github.com/zhangzhi45/script-writer-xhs.git
   ```

2. 在 WorkBuddy 中激活 skill：
   ```
   /use script-writer-xhs
   ```

3. 按提示输入主题/选题/需求即可开始创作。

## 账号风格

**人设**：AI 工具的深度使用者和研究者  
**语气**：像朋友间平静聊天，不夸张不套路  
**特点**：
- 口语化强，保留真人停顿感
- 逻辑清晰，言之有物
- 不虚构事实、不夸大效果
- 拒绝 AI 腔，保持真实感和人味

## 评分标准

| 维度 | 分值 | 检查点 |
|-----|------|--------|
| 选题价值 | 20 分 | 有痛点/反常识/情绪共鸣 |
| 开头吸引力 | 20 分 | 3 秒抓住注意力，有情绪 |
| 内容密度 | 20 分 | 无废话，信息增量≥70% |
| 风格匹配 | 20 分 | 符合麦克斯坦口语化风格 |
| 合规安全 | 20 分 | 无违禁词，无虚假宣传 |

**评分等级**：
- 🏆 **≥80 分**：可发布
- ⚠️ **60-79 分**：需优化
- ❌ **<60 分**：建议重写

## 一票否决

以下情况直接判定为不合格：
- 出现"谁懂啊/救命/绝了/太牛了/宝子们"等禁止词汇
- 开头无情绪，纯陈述事实
- 出现"最/第一/国家级"等绝对化用语
- 夸大效果、虚构数据
- "点赞收藏才告诉你"等诱导互动话术
- "作为 AI 模型/值得注意的是"等 AI 腔
