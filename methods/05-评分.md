# Method 05 — 量化评分

## 适用场景

- 用户说"评分"或"打分"
- 需要客观评估文案质量
- 给出具体改进建议

---

## 评分维度

| 维度 | 分值 | 权重 | 核心检查点 |
|-----|------|------|-----------|
| **选题价值** | 0-20分 | 20% | 有痛点/反常识/情绪共鸣 |
| **开头吸引力** | 0-20分 | 20% | 3秒抓住注意力，有情绪 |
| **内容密度** | 0-20分 | 20% | 无废话，信息增量≥70% |
| **风格匹配** | 0-20分 | 20% | 符合麦克斯坦口语化风格 |
| **合规安全** | 0-20分 | 20% | 无违禁词，无虚假宣传 |

**总分**：满分 100 分

---

## 评分等级

| 分数 | 等级 | 说明 | 建议 |
|-----|------|------|------|
| **90-100** | 🏆 优秀 | 可直接发布，有爆款潜力 | 保持水准，考虑多平台分发 |
| **80-89** | ✅ 良好 | 质量过关，可发布 | 微调后发布，关注数据反馈 |
| **70-79** | ⚠️ 合格 | 基本可用，建议优化 | 按改进建议修改后再发布 |
| **60-69** | ❌ 待改进 | 有明显问题 | 重点修改薄弱维度 |
| **<60** | 🚫 不合格 | 不建议发布 | 建议重写或换选题 |

---

## 一票否决项

以下情况直接判定为不合格（<60分）：

| 否决项 | 说明 |
|-------|------|
| **禁止词汇** | 出现"谁懂啊/救命/绝了/太牛了/宝子们"等 |
| **开头无情绪** | 纯陈述事实，无"惊到我了/忍不住"等情绪表达 |
| **违禁词** | 出现"最/第一/国家级/首选/顶级"等绝对化用语 |
| **虚假宣传** | 夸大效果、虚构数据、承诺疗效 |
| **诱导互动** | "点赞收藏才告诉你/关注后私聊"等诱导话术 |
| **AI腔** | 出现"作为AI模型/值得注意的是/综上所述" |

---

## 详细评分标准

### 1. 选题价值（0-20分）

| 分值 | 标准 |
|-----|------|
| 18-20 | 痛点精准，反常识，有强烈情绪共鸣，用户看了就想点 |
| 14-17 | 痛点明确，有一定新意，能引发兴趣 |
| 10-13 | 痛点一般，新意不足，但还算有用 |
| 6-9 | 痛点模糊，缺乏吸引力 |
| 0-5 | 无痛点，用户觉得"跟我没关系" |

**检查点**：
- [ ] 是否解决了用户的具体问题？
- [ ] 是否有反常识或新视角？
- [ ] 是否能引发情绪共鸣？

---

### 2. 开头吸引力（0-20分）

| 分值 | 标准 |
|-----|------|
| 18-20 | 3秒内抓住注意力，情绪强烈，用户忍不住继续看 |
| 14-17 | 开头有吸引力，能引发好奇心 |
| 10-13 | 开头一般，需要优化 |
| 6-9 | 开头平淡，难以抓住人 |
| 0-5 | 开头无趣，用户直接划走 |

**检查点**：
- [ ] 是否有情绪表达（"惊到我了/忍不住"）？
- [ ] 是否回应了"跟我有什么关系"？
- [ ] 是否点出了核心价值？

---

### 3. 内容密度（0-20分）

| 分值 | 标准 |
|-----|------|
| 18-20 | 信息密度极高，每句话都有价值，看完收获满满 |
| 14-17 | 信息丰富，有干货，少量可删减内容 |
| 10-13 | 信息一般，有部分废话或冗余 |
| 6-9 | 信息稀疏，废话较多 |
| 0-5 | 空洞无物，看完没什么收获 |

**检查点**：
- [ ] 每句话都有存在的理由吗？
- [ ] 删除任何一句话会影响表达吗？
- [ ] 能产生行为或认知改变吗？

---

### 4. 风格匹配（0-20分）

| 分值 | 标准 |
|-----|------|
| 18-20 | 完美匹配麦克斯坦风格，像本人写的 |
| 14-17 | 风格基本一致，少量地方可优化 |
| 10-13 | 风格尚可，有明显不匹配的地方 |
| 6-9 | 风格偏差较大，像别人写的 |
| 0-5 | 完全不符合，太书面或太AI |

**检查点**：
- [ ] 是否像朋友间平静聊天？
- [ ] 是否用了"其实/呢/吧"等语气词？
- [ ] 是否保留了真人停顿感？
- [ ] 是否避免了AI腔 and 套路表达？

---

### 5. 合规安全（0-20分）

| 分值 | 标准 |
|-----|------|
| 18-20 | 完全合规，无任何风险 |
| 14-17 | 基本合规，个别用词可优化 |
| 10-13 | 有小问题，需要修改 |
| 6-9 | 有明显违规风险 |
| 0-5 | 严重违规，不能发布 |

**检查点**：
- [ ] 无绝对化用语（最/第一/顶级）？
- [ ] 无虚假宣传或夸大效果？
- [ ] 无诱导互动话术？
- [ ] 无敏感内容（政治/色情/暴力/医疗）？

## 评分触发方式

用户说以下关键词时触发评分：
- "评分"
- "打分"
- "检查一下"
- "看看质量"
- "这个怎么样"
- "这个文案打几分"

---

## 评分后流程

1. **≥80分**：告知用户可以发布，建议关注数据反馈
2. **60-79分**：给出具体改进建议，修改后可再评分
3. **<60分**：建议重写或换选题，给出核心问题分析
