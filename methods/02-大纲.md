# Method 02 — 大纲生成

## 适用场景

- 已确定选题角度
- 需要生成结构化的内容大纲
- 用户确认后再撰写全文

---

## 执行流程

### Step 0：读取爆款索引（新增）

**生成大纲前，必须先读取 `Vault/爆款脚本索引.md`**

查看历史爆款内容的结构特点：

| 爆款标题 | 核心公式 | 结构特点 |
|---------|---------|---------|
| CoPAW开源Agent与Skills应用 | 痛点前置型工具介绍 | 痛点→解决方案→能力展示→场景演示 |
| WorkBuddy零门槛AI Agent体验 | 对比型工具推荐 | 痛点共鸣→对比维度→场景演示→行动号召 |

**借鉴思路**：
- 工具推荐类：用"痛点前置"或"对比型"结构
- 开头必须直击痛点（如"装不上""不会用"）
- 中间要有具体场景演示（数据分析、办公自动化）
- 结尾要有明确的互动引导

**大纲生成原则**：
- 优先使用爆款索引中验证过的结构
- 根据4A类型选择对应公式模板
- 在关键节点（开头/结尾）套用素材库句式

### Step 1：确定 4A 类型

根据选题性质，确认内容类型（已在 Method 01 中判断）：

| 类型 | 核心目标 | 大纲结构 |
|-----|---------|---------|
| **行动型** | 让观众"照着做" | 价值承诺→工具准备→分步操作→总结建议 |
| **分析型** | 帮观众"看清真相" | 现象引入→拆解分析→底层逻辑→启发总结 |
| **启发型** | 给予信念和能量 | 故事开场→转折冲突→反思感悟→行动建议 |
| **人类学型** | 给新认知镜头 | 现象描述→原因探究→新视角→思考留白 |

### Step 2：选择公式模板（新增）

根据 4A 类型和爆款索引，选择对应的公式：

| 4A类型 | 素材库对应公式 | 核心特点 | 参考爆款 |
|-------|--------------|---------|---------|
| 行动型 | 痛点前置型工具介绍 | 痛点→解决方案→能力展示→场景演示 | CoPAW |
| 行动型 | 对比型工具推荐 | 痛点共鸣→对比维度→场景演示→行动号召 | WorkBuddy |
| 分析型 | 行业洞察公式 | 现象引入→深度分析→趋势判断 | - |
| 启发型 | 经验分享公式 | 故事开场→转折→感悟 | - |
| 人类学型 | 认知升级公式 | 反直觉现象→原理剖析→新视角 | - |

**工具推荐类选题优先使用**：
- 「痛点前置型」（如CoPAW结构）
- 「对比型」（如WorkBuddy结构）

## 4A 大纲模板

### 行动型大纲

```
【开头】价值承诺（3-5秒抓住注意力）
  - 黄金前三秒：【核心】必须立刻前置演示最震撼的操作成果或直击痛点，绝对不能干讲！
  - 彻底删减"前情提要"（如：上一期我们讲了...）
  - 情绪钩子："我刚发现..."、"惊到我了..."
  - 价值点出：看完能得到什么

【第一段】核心亮点/王炸成果（前 15 秒完成价值交付）
  - 立刻展示最核心的爽点或高价值场景，防止由于"铺垫过长"导致流失
  - 讲清楚这个工具/方法最强大的地方，给路人一个留下来的理由

【第二段】工具/方法准备与分步操作（核心操作）
  - 需要用什么工具 / 前置条件
  - 步骤1：具体操作
  - 步骤2：具体操作
  - 步骤3：具体操作
  - （每步配合截图/演示说明）

【第三段】常见问题/注意事项
  - 可能遇到的卡点
  - 提前解答疑问

【结尾】总结建议 + 互动
  - 快速回顾核心步骤
  - 给出一个进阶建议
  - 开放式问题引导评论
```

### 分析型大纲

```
【开头】现象引入（制造认知冲突）
  - 具体现象或数据
  - 点出"大多数人没看懂的点"
  - 引发好奇心

【第一段】拆解分析
  - 深入剖析原因
  - 用对比、类比帮助理解
  - 展示不同角度

【第二段】底层逻辑
  - 揭示深层规律
  - 连接更大背景/趋势
  - 建立认知框架

【第三段】案例佐证
  - 具体例子说明
  - 数据或事实支撑

【结尾】启发总结 + 互动
  - 提炼有用认知
  - 给出行动建议
  - 开放性问题引发思考
```

### 启发型大纲

```
【开头】故事开场（建立代入感）
  - 具体场景或冲突
  - 点出当时的困境
  - "那时候我也..."

【第一段】转折冲突
  - 关键转折点
  - 当时的挣扎或选择
  - 情绪起伏

【第二段】反思感悟
  - 从中得到的领悟
  - 提炼普适性道理
  - 连接观众类似情况

【第三段】改变与结果
  - 具体的改变方法
  - 改变后的结果
  - 降低行动门槛

【结尾】行动建议 + 互动
  - 邀请分享类似经历
  - 传递鼓励和支持
  - 开放提问
```

### 人类学型大纲

```
【开头】现象描述（点出反直觉之处）
  - 普遍但未被注意的现象
  - 具体例子引发共鸣
  - "你有没有发现..."

【第一段】原因探究
  - 深层原因分析
  - 心理学/社会学角度
  - 隐藏的规律或机制

【第二段】新视角
  - 全新的解释框架
  - 重新理解熟悉事物
  - 连接更大背景

【第三段】延伸思考
  - 相关现象的联系
  - 不同场景的应用

【结尾】思考留白 + 互动
  - 开放性问题
  - 不给标准答案
  - 邀请分享观点
```

## 大纲质量检查

| 检查项 | 合格标准 | 一票否决 |
|-------|---------|---------|
| 有"为什么" | 明确解释为什么要看/做 | ❌ 无则不合格 |
| 理由数量 | ≥3个核心要点 | ❌ <3个不合格 |
| 用户覆盖 | 能覆盖≥2类用户 | ❌ <2类不合格 |
