## 快速检查清单

### 内容质量
- [ ] 开头有情绪（"惊到我了/忍不住"）
- [ ] **黄金3秒钩子**：无前情提要，直接展示最震撼成果或直击痛点
- [ ] 回应3个问题（跟我有什么关系/为什么听你的/能得到什么）
- [ ] 无禁止词汇（谁懂啊/救命/绝了/太牛了）
- [ ] **高点击标题**：包含具体收益词，不只有技术名词
- [ ] 信息密度高（每句话都有价值）
- [ ] 风格口语化（像朋友聊天）
- [ ] 结尾有互动

### 平台合规
- [ ] 无绝对化用语（最/第一/顶级）
- [ ] 无虚假宣传
- [ ] 无诱导互动（"点赞收藏才告诉你"）
- [ ] 无敏感内容

## 一票否决

以下情况直接判定为不合格：

| 否决项 | 说明 |
|-------|------|
| 禁止词汇 | 出现"谁懂啊/救命/绝了/太牛了/宝子们"等 |
| 开头无情绪 | 纯陈述事实，无情绪表达 |
| 违禁词 | 出现"最/第一/国家级"等绝对化用语 |
| 虚假宣传 | 夸大效果、虚构数据 |
| 诱导互动 | "点赞收藏才告诉你"等话术 |
| AI腔 | "作为AI模型/值得注意的是"等 |

**如有任何一项，直接打回修改。**

## 评分标准

用户说"评分"时，按以下5维度打分：

| 维度 | 分值 | 检查点 |
|-----|------|--------|
| 选题价值 | 20分 | 有痛点/反常识/情绪共鸣 |
| 开头吸引力 | 20分 | 3秒抓住注意力，有情绪 |
| 内容密度 | 20分 | 无废话，信息增量≥70% |
| 风格匹配 | 20分 | 符合麦克斯坦口语化风格 |
| 合规安全 | 20分 | 无违禁词，无虚假宣传 |

**评分等级**：
- 🏆 **≥80分**：可发布
- ⚠️ **60-79分**：需优化
- ❌ **<60分**：建议重写

## 核心原则

1. **先读样本文稿**：撰写前必须读取 `references/tone-samples/`
2. **先确认大纲**：用户确认大纲后再撰写全文
3. **该长则长，该短则短**：一切服务于内容本身
4. **没有废话**：每句话都应该有存在的理由
